Website Rental Mobil

Web Development, JavaScript, Rental Management System Completed: May 2024

Project Highlights

  • Online booking system with real-time car availability checking.
  • Admin dashboard for managing rentals, verifying payments, and generating reports.
  • User features including registration, login, profile management, and rental history tracking.
  • Fully responsive design with intuitive navigation and real-time data validation.

About the Project

The Web-Based Car Rental System is a digital solution developed to simplify the car rental process at Ian Transport, a transportation service company based in Yogyakarta. This system addresses the inefficiencies of the traditional manual rental method, which often caused delays and human errors.

With this web-based platform, customers can easily browse available vehicles, check availability, make reservations, and upload proof of payment‚Äîall online. On the other hand, the admin is equipped with tools to manage rental transactions, vehicle data, drivers, users, and generate rental reports. The system is developed using PHP and MySQL, with a responsive and user-friendly interface powered by HTML, CSS, and Bootstrap.

Technical Implementation

The system is built using HTML, CSS, Bootstrap, JavaScript, and PHP, and implements a MySQL relational database for full data management. Here's a breakdown of its main components:

  • Frontend: Developed with HTML and CSS, enhanced with Bootstrap for responsive layouts and modern UI
  • Backend: PHP is used to handle application logic, authentication, form submissions, and database operations
  • Database: MySQL is used to store all critical data including cars, users, bookings, drivers, and transactions
  • Validation & Testing: Black-box testing is implemented to ensure functionality works according to user requirements

Challenges & Solutions

One of the primary challenges was transitioning from a manual rental process to a fully digital platform. Previously, customers had to contact Ian Transport via WhatsApp or visit the office to check availability and confirm bookings. This was time-consuming and prone to error. The proposed solution was to develop a centralized web-based system that automates car availability checking, booking, payment validation, and report generation‚Äîimproving efficiency and user experience for both customers and the company.
