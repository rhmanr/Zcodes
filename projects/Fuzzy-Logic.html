Fuzzy-Based Anxiety Level Assessment System for New Migrant Students

Fuzzy Logic, Expert System, Web Application Development Completed: June 2025

Project Highlights

  • Developed a Fuzzy Tsukamoto system to assess anxiety based on questionnaire data.
  • Converted linguistic responses (e.g., "Sometimes", "Often") into numeric scores.
  • Designed custom membership functions and fuzzy inference rules.
  • Built an interactive web app using Streamlit for real-time result visualization.

About the Project

Fuzzy-Based Anxiety Level Assessment System for New Migrant Students is a fuzzy logic-based application designed to evaluate the anxiety levels of new migrant students. It uses the Fuzzy Tsukamoto inference method, where each response from a psychological questionnaire is converted into numerical values, processed through membership functions, and then inferred into one of three output levels: Low, Moderate, or High anxiety.

With an interactive web interface built using Streamlit, the system is designed to be accessible, lightweight, and easy to use for students, counselors, or institutions aiming to perform early screening of emotional and psychological conditions among students adapting to a new environment.

Technical Implementation

The system is built using Python and implements the Fuzzy Tsukamoto inference technique to process questionnaire responses. Here's a breakdown of its main components:

  • Fuzzy Tsukamoto Logic: Calculates output anxiety levels based on fuzzy rules and numeric input derived from user responses
  • Membership Functions: Define the degree of membership for each input value in categories like Low, Medium, and High
  • Fuzzy Rules: A rule base is constructed to handle different combinations of user responses and map them to appropriate output levels
  • Python & NumPy: Handle numerical calculations and data preprocessing
  • Matplotlib: Used to visualize outputs such as charts and scores
  • Streamlit: Acts as the frontend web interface for user input and result display
