Handwriting Recognition of 2024 Presidential Election Results

Optical Character Recognition (OCR), Deep Learning, Computer Vision Completed: June 2025

Project Highlights

  • 95%+ Accuracy achieved using HOG features with SVM classifier.
  • Multi-Algorithm Comparison across 5 different machine learning models.
  • Real-time Processing with efficient batch processing for large datasets.
  • 3 Feature Extraction Methods implemented and compared.

About the Project

This project demonstrates a comprehensive approach to handwritten digit recognition, combining traditional machine learning techniques with modern software engineering practices. The system provides an end-to-end solution from raw image processing to model evaluation, wrapped in an intuitive desktop application.

The project is designed to recognize handwritten Sundanese characters, which are essential for preserving and promoting the Sundanese language and culture. The system allows users to upload images of handwritten characters, which are then processed and classified using a trained machine learning model.

Technical Implementation

The system is built using Python with the following key technologies:

  • Machine Learning Classification: Multiple algorithms including SVM, Random Forest, and KNN for digit recognitions.
  • TensorFlow: Used for the neural network components
  • OpenCV: For image preprocessing and augmentation
  • Scikit-learn: For model evaluation and metrics
  • Matplotlib/Seaborn: For visualizing training results and model performance
  • Jupyter Notebook: For interactive development and testing
  • Split Dataset: To evaluate model generalization and documentation
  • Streamlit: Provides the web interface for interacting with the system

Challenges & Solutions

One of the main challenges in this project was handling inconsistent image formats and sizes from different sources, which could significantly impact model performance. To overcome this, I implemented a robust dual-loading system using both OpenCV and PIL libraries, ensuring reliable image processing regardless of format corruption or compatibility issues. The system automatically standardizes all images to 64√ó64 grayscale format with proper normalization.

Another challenge was achieving high accuracy in character recognition. I experimented with various feature extraction techniques, including Histogram of Oriented Gradients (HOG) and Principal Component Analysis (PCA), to enhance model performance. The final model achieved over 95% accuracy using HOG features with a Support Vector Machine (SVM) classifier.

Additionally, I implemented a batch processing system to handle large datasets efficiently, allowing for real-time processing of multiple images without significant performance degradation.
