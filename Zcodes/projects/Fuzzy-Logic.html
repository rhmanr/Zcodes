About the Project

This project explores the use of deep learning, particularly Convolutional Neural Networks (CNN), to recognize handwritten characters from the Sundanese script.

By applying Computer Vision techniques, the system can classify individual characters from image inputs, contributing to the development of Optical Character Recognition (OCR) systems for non-Latin scripts. Beyond technical implementation, this project also supports the digital preservation and recognition of Indonesia‚Äôs traditional writing systems.

Technical Implementation

The system is built using Python with the following key technologies:

  • Convolutional Neural Networks (CNN): For classifying handwritten Sundanese characters.
  • TensorFlow: Used for the neural network components
  • OpenCV: For image preprocessing and augmentation
  • Scikit-learn: For model evaluation and metrics
  • Matplotlib/Seaborn: For visualizing training results and model performance
  • Jupyter Notebook: For interactive development and testing
  • Split Dataset: To evaluate model generalization and documentation
  • Streamlit: Provides the web interface for interacting with the system

Challenges & Solutions

One of the main challenges in this project was the lack of publicly available Sundanese handwriting datasets, which are essential for training deep learning models. To overcome this, I created a custom dataset by collecting handwritten samples from my own writing, ensuring sufficient variation in stroke styles and character forms. Another challenge was maintaining consistent data quality, especially during the preprocessing of handwritten inputs that varied in thickness, noise, and alignment. This issue was addressed using image preprocessing techniques with OpenCV, including grayscale conversion, resizing, and normalization. Finally, achieving high model accuracy required careful hyperparameter tuning and the application of data augmentation techniques to improve generalization and prevent overfitting.
